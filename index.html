Skill Swap — Full-Stack Web Application

Python Flask Pandas NumPy JavaScript PostgreSQL REST APIs

OVERVIEW

Skill Swap is a collaborative peer-to-peer learning platform that connects learners and mentors for real-time skill exchange. Built as a full-stack web application, it provides an intuitive interface for users to list, discover, and share skills across multiple domains.

KEY FEATURES

  • ▹Developed a full-stack web application using HTML, CSS, and JavaScript for the frontend and Flask (Python) for the backend, following the MVC architecture pattern for clean separation of concerns.

  • ▹Implemented real-time chat, video conferencing, and community forums to enhance collaborative learning experiences, enabling users to communicate and learn together seamlessly.
  • ▹Leveraged Pandas and NumPy for building analytics dashboards that track user engagement, skill popularity, and learning progress metrics.

  • ▹Implemented secure user authentication and session management using Flask-Login, with password hashing (bcrypt) and CSRF protection for secure form submissions.
  • ▹Built a notification system to alert users about new skill matches, incoming chat messages, and scheduled video conferencing sessions.

Diabetic Retinopathy — Explainable AI Diagnosis

TensorFlow Keras OpenCV Scikit-learn Cloud Deploy

OVERVIEW

Developing a transparent diagnosis model for Diabetic Retinopathy using Explainable AI. This project addresses the critical need for trustworthy AI in healthcare by combining high-accuracy deep learning with interpretable visual explanations, enabling clinicians to understand and validate AI-driven diagnoses.

KEY FEATURES

  • ▹Developed a deep learning model (CNN) for automated detection of diabetic retinopathy from retinal fundus images, achieving multi-class severity grading (No DR, Mild, Moderate, Severe, Proliferative).
  • ▹Applied Explainable AI (XAI) techniques (e.g., Grad-CAM, LIME, SHAP) to improve transparency and interpretability of predictions, generating visual heatmaps that highlight regions of interest on retinal scans.
  • ▹Implemented data preprocessing and augmentation (rotation, flips, brightness adjustment) to improve generalization and handle class imbalance across different severity levels.
  • ▹Utilized TensorFlow/Keras for model architecture with transfer learning from pre-trained networks (InceptionV3), fine-tuned on domain-specific retinal image datasets.
  • ▹Integrated OpenCV for advanced image preprocessing — including noise reduction, contrast enhancement (CLAHE), and retinal vessel segmentation for improved feature extraction.
  • ▹Built a comprehensive evaluation framework using Scikit-learn with precision, recall, F1-score, confusion matrix, and AUC-ROC curve analysis to validate model performance against clinical benchmarks.
  • ▹Deployed the model on a cloud platform with a web-based interface, allowing clinicians to upload retinal images and receive instant diagnoses with visual explanations and confidence scores.
